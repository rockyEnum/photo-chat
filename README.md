# 证件照小程序

一款基于 Uni-app + Vue3 + TypeScript 开发的智能证件照制作小程序。

## 功能特性

- 📸 支持拍照和相册选择照片
- 📏 提供6种常用证件照尺寸（一寸、二寸、小一寸、大一寸等）
- 🎨 6种背景色可选（白、蓝、红、浅蓝、浅灰、深蓝）
- ✂️ 支持图片缩放和位置微调
- 📐 自动排版功能（1寸9张、2寸4张排版）
- 💾 高清导出，支持保存到相册

## 技术栈

- 框架：Uni-app (Vue3 + TypeScript)
- 平台：微信小程序（可扩展至支付宝、抖音等）
- 图片处理：Canvas 2D API
- 构建工具：Vite

## 项目结构

```
├── pages/
│   ├── index/           # 首页-尺寸选择
│   ├── camera/          # 拍照/上传
│   ├── edit/            # 编辑处理
│   └── result/          # 结果预览
├── utils/
│   ├── constants.ts     # 尺寸、颜色常量
│   ├── imageProcessor.ts # 图片处理工具
│   └── auth.ts          # 权限管理
├── App.vue              # 应用入口
├── main.ts              # 主入口
├── manifest.json        # 应用配置
├── pages.json           # 页面配置
└── vite.config.ts       # Vite配置
```

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 开发模式

```bash
npm run dev:mp-weixin
```

### 3. 构建发布

```bash
npm run build:mp-weixin
```

## 证件照尺寸规格

| 名称 | 像素尺寸 | 毫米尺寸 | 用途 |
|------|---------|---------|------|
| 一寸 | 295×413 | 25×35mm | 简历、证书 |
| 二寸 | 413×626 | 35×49mm | 证书、护照 |
| 小一寸 | 260×378 | 22×32mm | 驾驶证、体检 |
| 大一寸 | 390×567 | 33×48mm | 护照、签证 |
| 小二寸 | 334×472 | 28×40mm | 公务员考试 |
| 大二寸 | 413×531 | 35×45mm | 护照 |

## 背景色选项

- 白色 (#FFFFFF) - 通用
- 蓝色 (#438EDB) - 简历、毕业证
- 红色 (#D9534F) - 结婚证、证件照
- 浅蓝 (#87CEEB) - 护照、身份证
- 浅灰 (#F5F5F5) - 签证
- 深蓝 (#1E3A5F) - 毕业证

## 权限说明

小程序需要以下权限：

- `scope.camera` - 用于拍摄证件照
- `scope.writePhotosAlbum` - 用于保存照片到相册

## 注意事项

1. 小程序基础库要求 >= 2.19.0
2. 如需集成腾讯云AI人像分割，需要配置服务端API
3. 首次使用会引导用户授权相机和相册权限
